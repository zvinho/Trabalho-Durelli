Sobre o Projeto
  Este projeto consiste em um conversor de bases numéricas desenvolvido como parte do Trabalho Prático 1.
  O objetivo é implementar manualmente os algoritmos de conversão entre os sistemas Decimal, Binário, Octal e Hexadecimal, sem utilizar funções prontas da linguagem.

Linguagem Escolhida:
  C++ 

Link do vídeo: https://youtu.be/wQL9IWMzsOs?si=B436yDLfoVk-O9jp

Instruções de execução:
  Ao executar, será solicitado o metodo de conversão, sendo eles: 
  1- Conversão manual, onde você informa um valor, a base desse valor e escolhe para qual base você deseja converter esse valor;
  2- Modo batch, onde você seleciona um arquivo "entrada.csv" com os dados para conversão e os resultados são gravados em um arquivo "saida.csv"; 
  3- Quiz, onde você inicia um jogo de perguntas e respostas sobre conversões de bases numéricas;
  4- Calculadora de máximos, onde você informa uma quantidade de bits e é retornado o valor máximo em cada sistema numérico;
  
  Para a conversão manual (1), informe qual número você quer converter. Em seguida você irá informar em qual base esse valor informado está, sendo 2 para binário, 8 para octal, 10 para decimal e 16 para hexadecimal, digite apenas o número referente à base. Em seguida será solicitado para qual base você quer converter o valor informado no primeiro passo, novamente seguindo a ideia de 2, 8, 10 e 16. O programa irá converter o valor informado na base informada para a base desejada.
  Para o modo batch (2), você precisa ter um arquivo "entrada.csv" com valores informados na seguinte ordem: valor, base de origem e base de destino. Os resultados são gravados no arquivo "saida.csv" na seguinte ordem: valor, base de origem, resultado e base do resultado;
  Para o quiz (3), você irá responder uma série de 5 perguntas sobre conversão entre sistemas numéricos e ao final será informado quantas você acertou.
  Para a calculadora de máximos (4) você irá informar uma quantidade de bits e e será retornado o valor máximo dessa quantidade em cada sistema numérico.
