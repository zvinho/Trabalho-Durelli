Sobre o Projeto
  Este projeto consiste em um conversor de bases numéricas desenvolvido como parte do Trabalho Prático 1.
  O objetivo é implementar manualmente os algoritmos de conversão entre os sistemas Decimal, Binário, Octal e Hexadecimal, sem utilizar funções prontas da linguagem.

Linguagem Escolhida:
  C++ 

Instruções de execução:
  Ao executar, será solicitado um valor para conversão, informe qual número vc quer converter. Em seguida você irá informar em qual base esse valor informado está, sendo 2 para binário, 8 para octal,
  10 para decimal e 16 para hexadecimal, digite apenas o número referente à base. Em seguida será solicitado para qual base você quer converter o valor informado no primeiro passo, novamente seguindo
  a ideia de 2, 8, 10 e 16. O programa irá converter o valor informado na base informada para a base desejada.
